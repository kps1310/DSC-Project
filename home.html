MONITORING SOCIAL DISTANCING

ML Model to monitor social distancing using Python & OpenCV

Tool to monitor social distancing from CCTV, videos using Python, Deep learning, Computer Vision. This tool can automatically estimate interpersonal distance from uncalibrated RGB cameras. Can be used at public places and workplace.In the fight against the COVID-19, social distancing has proven to be a very effective measure.
To ensure social distancing protocol in public places and workplace, I have developed social distancing detection tool that can monitor if people are keeping a safe distance from each other by analyzing real time video streams from the camera. This tool has following features:
Detect humans in the frame with yolov3.
Calculates the distance between every human who is detected in the frame.
Shows how many people are at High, Low and Not at risk

  • Category : ML Model

ENHANCED AES ALGORITHM

Parallel Implementation of AES algorithm using CUDA

AES is the most widely used encryption algorithm in various applications. One of its main implementations is in the SSL and TLS protocols. In this project, I focused on the parallel implementation of AES providing flexibility and performance improvement in terms of acceleration.
It concluded that the parallel version of AES is almost 4-5 times faster as compared to the vanilla version. The runtime majorly depends upon the size of the input file, the difference between the execution time of the two algorithms widens as we increase the file size.

  • Category: Network Information Security
